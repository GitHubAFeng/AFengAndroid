# Usage

copy SunBabyLoadingView.java文件 放入自己的项目当中, 在布局文中引用即可

    <com.zuck.swipe.sunbaby.view.SunBabyLoadingView
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"/>
